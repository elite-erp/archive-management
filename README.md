# Erp Archive

## Install
1- install dependances
```bash
composer install
npm i -g pnpm
pnpm i
```
2- copy env file
```bash
cp .env.example .env
```
3- setup database as needed (mysql or postgresql)

## Update
```bash
composer update
pnpm i
```

## Start Development
```bash
php artisan serve
pnpm dev
```
